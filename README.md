### 一、相关技术
- 后端：`Java`、`JavaWeb` / `Springboot`。
- 前端：`Vue`、`HTML` / `CSS` / `Javascript` 等。
- 数据库：`MySQL`

- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- 等

### 三、功能描述
系统分为（身份）：普通用户、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 首页
4. 景点信息
5. 路线推荐
6. 论坛交流
7. 新闻资讯
8. 个人中心
9. 我的发布
10. 我的收藏


**管理员功能：**
1. 登录
2. 个人中心（修改密码、个人信息）
3. 用户管理
4. 景点类型管理
5. 景点信息管理
6. 线路推荐管理
7. 预订信息管理
8. 论坛交流
9. 系统管理（轮播图管理、新闻资讯）
